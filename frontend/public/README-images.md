# Instructions pour ajouter les images de cartes

## Étapes à suivre :

1. **Remplacer les fichiers placeholders :**
   - Remplacez `cards-sample-1.png` par la première image de cartes que vous avez fournie
   - Remplacez `cards-sample-2.png` par la deuxième image de cartes que vous avez fournie

2. **Où placer les images :**
   - Les images doivent être placées dans le dossier `frontend/public/`
   - Les noms de fichiers doivent rester exactement : 
     - `cards-sample-1.png`
     - `cards-sample-2.png`

3. **Format recommandé :**
   - Format : PNG
   - Taille : optimisée pour le web (compression 80-90%)
   - Dimensions : recommandé 1200px de largeur maximum

## Modifications effectuées sur la landing page :

✅ **Logo Kartaro :**
- Centré et agrandi (60px sur mobile, 80px sur desktop)
- Ajouté au-dessus du titre principal

✅ **Section exemples de cartes :**
- Ajoutée après la section "Personnalisation"
- Deux colonnes responsives
- Images avec bordures arrondies et ombres
- Textes descriptifs pour chaque thème

Les images s'afficheront correctement une fois les fichiers remplacés.

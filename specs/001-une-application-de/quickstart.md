# Quickstart

1. **Start the backend server**

   ```bash
   cd backend
   npm install
   npm start
   ```

2. **Start the frontend server**

   ```bash
   cd frontend
   npm install
   npm start
   ```

3. **Open the application in your browser**

   [http://localhost:3000](http://localhost:3000)

4. **Generate cards**

   - Enter a theme and context in the input fields.
   - Click the "Generate" button.
   - The generated cards will be displayed on the screen.

5. **Customize a card**

   - Click on a card to open the editor.
   - Modify the title, description, or icon.
   - Click the "Save" button.

6. **Create a collection**

   - Select the cards you want to include in the collection.
   - Click the "Create Collection" button.
   - Enter a name for the collection and click "Save".
